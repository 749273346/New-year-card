# Stripe

## 什么是 Stripe？
Stripe 是全球最流行的在线支付处理平台之一，为互联网业务提供了一整套支付基础设施。它以开发者友好的 API 和极佳的用户体验著称。

## 主要功能
- **全球支付**：支持 135+ 种货币和各种支付方式（信用卡、支付宝、微信支付、Apple Pay 等）。
- **订阅管理**：轻松实现会员制、定期扣款等复杂的订阅逻辑。
- **安全合规**：处理所有敏感的支付信息和合规性问题（如 PCI DSS），让开发者无需担心安全风险。
- **Checkout 页面**：提供托管的支付页面，几行代码即可接入美观的收银台。

## 在本项目中的作用
如果我们的“新年贺卡”应用需要**收费功能**（例如：生成高级版贺卡、去除水印、订阅会员），Stripe 将负责处理所有的**支付环节**，确保钱款安全到账。
