# Vercel

## 什么是 Vercel？
Vercel 是一个专为前端框架（特别是 Next.js）优化的云平台，旨在让开发者能够快速构建、预览和部署 Web 应用。它不仅支持静态网站托管，还完美支持服务端渲染 (SSR) 和边缘计算 (Edge Functions)。

## 主要功能
- **零配置部署**：只需连接 Git 仓库（如 GitHub、GitLab），每次代码推送都会自动触发构建和部署。
- **全球边缘网络**：利用全球 CDN 节点，让你的应用在世界各地都能极速访问。
- **Serverless Functions**：无需管理服务器即可运行后端逻辑，按需自动扩缩容。
- **预览环境**：为每个 Pull Request 自动生成唯一的预览链接，方便团队协作和测试。

## 在本项目中的作用
我们将使用 Vercel 来**托管和部署**我们的“新年贺卡”应用。它将负责把你的 Next.js 代码变成用户可以访问的网站。
