# OpenRouter

## 什么是 OpenRouter？
OpenRouter 是一个**统一的 AI 模型聚合接口**。它不像 OpenAI 或 Google 那样自己训练模型，而是将市面上几乎所有优秀的开源和闭源模型（包括 GPT-4, Claude 3, Llama 3, Mistral 等）整合到一个 API 标准下。

## 主要功能
- **一站式访问**：只需要一个 API Key，就可以调用几十种不同的 AI 模型。
- **价格透明**：直观展示各个模型的价格，方便开发者选择性价比最高的模型。
- **无缝切换**：在代码中切换模型通常只需要改一个字符串，无需重写集成代码。
- **访问受限模型**：让开发者能够轻松访问一些在特定地区可能受限或部署困难的模型。

## 在本项目中的作用
OpenRouter 是**集成 AI 的“万能插头”**。使用它，我们可以在开发过程中灵活测试不同的模型，找到生成祝福语效果最好且成本最低的那个，而不需要分别为每个厂商注册账号和配置 API。
