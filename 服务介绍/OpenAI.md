# OpenAI

## 什么是 OpenAI？
OpenAI 是全球领先的人工智能研究实验室，开发了著名的 GPT 系列模型（如 GPT-3.5, GPT-4）。它们提供的 API 让开发者能够将强大的自然语言处理能力集成到自己的应用中。

## 主要功能
- **文本生成**：撰写文章、代码、故事、对话等（ChatGPT 背后的技术）。
- **图像生成**：通过 DALL·E 模型根据文本描述生成图片。
- **语音转文本**：通过 Whisper 模型进行高精度的语音识别。
- **嵌入 (Embeddings)**：用于语义搜索和知识库构建。

## 在本项目中的作用
在“新年贺卡”应用中，OpenAI 的 GPT 模型可以作为**核心 AI 引擎**，负责根据用户的简短描述，生成富有创意和个性化的新年祝福语。
